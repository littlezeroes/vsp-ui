🚪 4. Entry & Exit Points

Entry points (vào Sinh lời)

1. Home → Widget Sinh lời (GĐ1 — primary)

2. Tab Tài chính → Swipe card Sinh lời (GĐ2+)

3. Deep link từ push notification (trả lãi tháng)

Exit points (rời Sinh lời)

1. Back button → quay Home / Tab trước đó

2. S4/S10/S16 thành công → CTA "Về trang chủ"

3. Huỷ sinh lời (S16) → Trang chủ (feature bị xoá khỏi nav)
